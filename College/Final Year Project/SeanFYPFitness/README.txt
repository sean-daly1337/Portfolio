How to set up project

1. Open android studio.
2. Import this project by clicking "Open an existing project"
3. Navigate to where you saved this project on your computer, select it and then click open.
4. Once the project is opened, you will see it pop up as "app" on the project directory along the left.
5. Click "Build" on the top of the screen and then click "Make Project"
6. Once the project is built, select your emulator on the top right of the screen.
7. Once the emulator is selected, click the green play button to the right of it.
8. You should see your emulator loading up.
9. A splash screen saying "SeanFYPFitness" should be displayed at this point.
10. Finally, login and enjoy the application!
